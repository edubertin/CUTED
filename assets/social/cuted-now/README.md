# CUTED Now Social Assets

This folder contains visual assets for the CUTED Now TikTok channel.

## Approved

Use files under `approved/` for production profile and post work.

Current approved avatar:

```text
assets/social/cuted-now/approved/avatar.png
```

## Drafts

Use `drafts/` for contact sheets, experimental variants, and rejected visual
directions. Draft assets should not be uploaded to TikTok unless explicitly
approved.

## Notes

- The approved avatar is the clean version without diagonal background texture.
- Avoid the older blue/red `CE/D` style avatar for CUTED Now.
- Avoid gray watermark slabs; use transparent black capsules instead.

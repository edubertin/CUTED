# Changelog

As mudancas relevantes do CUTED serao registradas aqui.

## Unreleased

- Proximas correcoes e melhorias ainda nao publicadas.

## 2026.07.17 Beta 1

- Repositorio de codigo aberto ao publico sob AGPL-3.0.
- Primeiro instalador Windows publicado como prerelease unsigned.
- Site publico do CUTED publicado com download versionado pelo GitHub Releases.
- API localhost protegida por sessao local e validacao de `Host` e `Origin`.
- Documentacao de seguranca, privacidade, contribuicao e licencas.
- CI com permissoes minimas, Actions pinadas, Dependabot, CodeQL e Gitleaks.
- Instalador acompanhado por checksum SHA-256 e aviso sobre SmartScreen.

## 2026.07 Development Milestone

Este marco descreve o trabalho acumulado de julho. A primeira publicacao
binaria desse ciclo e a tag `v2026.07.17-beta.1`.

- Aplicativo desktop Windows com pywebview/WebView2.
- Build portatil PyInstaller e instalador Inno Setup.
- Projetos recentes e recuperacao de importacoes em andamento.
- Legendas em portugues e ingles sob demanda.
- Fila de render com perfis de recurso.
- Diagnosticos locais sanitizados.

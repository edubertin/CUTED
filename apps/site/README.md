# CUTED Public Site

Public product page for the CUTED Windows application, hosted with OpenAI
Sites. The page uses approved brand assets, real product screenshots, a
versioned GitHub Release download, and the CUTED Now TikTok case study.

## Local development

```powershell
npm ci
npm run dev
```

## Validation

```powershell
npm test
npm run lint
```

GitHub Releases remains the only authority for installer binaries. Do not add
installers, user media, project outputs, secrets, or local environment files to
this directory.

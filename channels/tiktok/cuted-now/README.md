# CUTED Now TikTok Operations

CUTED Now is the TikTok channel for testing timely short-form clips made with
CUTED. This folder tracks the channel operation separately from product specs,
render samples, and application code.

## Account

- URL: https://www.tiktok.com/@cutednow
- Handle: `@cutednow`
- Display name: `CUTED Now`
- Bio: `O agora em cortes rapidos. Feito com CUTED.`
- Current status: active public channel with published CUTED-made clips.
- Public case-study evidence observed on 2026-07-17: one clip exceeded 47,000
  views. Treat this as a timestamped observation, not a live metric.

## Operating Rules

- Keep account changes separate from product code changes.
- Do not publish, delete, promote, or reply without explicit approval.
- Every post should map to a source, pillar, hook, asset template, and metric row.
- Use current trend research only as directional input; do not chase trends that
  break channel identity or rights constraints.

## First Implementation Path

1. Build a 10-post backlog.
2. Render one approved test clip through CUTED.
3. Prepare caption and hashtags.
4. Get explicit approval before upload or publish.
5. Record 1h, 24h, and 7d metrics.
